<h2>Description</h2>
<h3>General information:</h3>
<ul>
  <li>
    A beautifully addictive puzzle game about connecting the dots.
  </li>
   <li>
    The goal is simple: connect same-colored dots vertically and horizontally to win points.
   </li>
  <li>
    This game is endless.
   </li>
</ul>

</ul>
